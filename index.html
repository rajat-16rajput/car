CAR stands for "Central Application Repository". It is a Centrailzed application which tracks all Deluxe Corp's Products.

It has 3 User Accessibilities - Admin, Member, Viewer.

Admin

The Admin persona has access to all CRUD operations

Member

The Member persona has access to only specific services on which he can perform CRUD operations. He can only perform edit operation when he is the owner of a specific Inventory

Viewer

The Viewer persona has can only read the data. He can't update or add new data in the application
